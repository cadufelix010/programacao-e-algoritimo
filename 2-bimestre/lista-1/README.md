## Como testar

Você pode copiar o conteúdo de cada arquivo e testar no compilador online:

https://www.programiz.com/cpp-programming/online-compiler/

Também é possível compilar pelo terminal com o g++:

```bash
g++ exercicio_01_primo.cpp -o exercicio_01
./exercicio_01
```

No Windows:

```bash
g++ exercicio_01_primo.cpp -o exercicio_01.exe
exercicio_01.exe
```

## Entrega no GitHub

1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta.
3. Inclua este README.md.
4. Teste cada programa antes da entrega.
